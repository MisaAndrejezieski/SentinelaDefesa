# 🛡️ SENTINELA - Sistema de Defesa Cibernética

Um sistema educacional para detecção de cryptojacking e monitoramento de segurança.

## 📋 Sobre o Projeto

O Sentinela é uma ferramenta **educacional** que ajuda a entender e detectar atividades suspeitas de mineração de criptomoedas (cryptojacking) em seu computador.

**⚠️ IMPORTANTE**: Este software é para fins EDUCACIONAIS. Use apenas em seus próprios sistemas.

## ✨ Funcionalidades

- 🔍 **Detecção de Processos**: Identifica processos suspeitos de mineração
- 📊 **Monitoramento de Recursos**: Acompanha uso de CPU e memória em tempo real
- 🚨 **Sistema de Alertas**: Notifica quando atividades anormais são detectadas
- 📝 **Geração de Logs**: Salva relatórios para análise posterior
- 🍯 **Honeypot Simples**: Simula servidores vulneráveis para estudo (opcional)

## 🚀 Como Instalar

### Pré-requisitos
- Windows 10/11
- Python 3.7 ou superior
- Git (para clonar o repositório)

### Passo a Passo

1. **Clone o repositório**
   ```bash
   git clone https://github.com/seu-usuario/sentinela-defesa.git
   cd sentinela-defesa
   ```

2. **Instale as dependências**
   ```bash
   pip install -r requirements.txt
   ```

3. **Execute o programa**
   ```bash
   python main.py
   ```
